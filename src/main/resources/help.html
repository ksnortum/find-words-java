Overview

Top

Scrabble Words will help you find words to play in the game of Scrabble. You enter letters of the tiles you have in your hand. Optionally, you can enter a letter or letters on the board that the suggested word must contain. You can also enter the letters the suggested word must start with and end with. Two dictionaries are supported: the Official Tournament and Club Word List, or TWL, (used in the USA) and SOWPODS (used in most other countries).

If you are familiar with Regular Expressions you can use them to limit your search. Options can be combined except Contains Letters and Contains Regex, which are mutually exclusive.

A list of suggested words is displayed when you press submit. All entered data is cleared when the clear button is pressed. The suggested words are displayed first by highest value first and then if the values are the same, by alphabetical order.

ScrabbleWords has a limited ability to detect whether a word is a bingo (with an extra 50 points). Specifically, it may fail to catch bingos if Contains Letters has more than one letter or if regular expressions are using. This means it is possible (but not likely) that a bingo word may not display first in the suggested word list.

Available Letters

Top

Enter the letters of the tiles in your hand. If you have a blank tile, enter this as a dot (.). Currently, the program cannot handle the situation where the player has both black tiles.

Contains Letters / Regex

Top

Select either Contains these letters or Contains (with regex).

Contains Letters

Select the first option if you are only going to enter a letter or letters that the suggested word must contain. This would be a letter on the board that the suggested word will intersect. If you enter more than one letter, the suggested word must contain all letters, not necessarily in order or next to each other. To get letter(s) in exact positions in the words, use a regular expression.

Contains Regex

Select the second option if know regular expressions and would like the suggested words to match this instead of simple letters. The regex will be tested to make sure it is a valid Java regular expression. Note that you will have to add the letters from the regex to Available Letters.

Starts With

Top

Enter the letter or letters the suggested words should start with. Do not enter these letters in Available Letters.

Ends With

Top

Enter the letter or letters the suggested words should end with. Do not enter these letters in Available Letters.

Dictionary

Top

Select the dictionary to use. Currently you have the choice between the Official Tournament and Club Word List, or TWL, (used in the USA) and SOWPODS (used in most other countries).
